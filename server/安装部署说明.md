# AI Image Studio - 安装部署说明

## 环境要求

| 软件 | 最低版本 | 说明 |
|------|---------|------|
| Node.js | >= 18 | 推荐使用 v22.x LTS |
| MongoDB | >= 5.0 | 数据库服务，需提前安装并运行 |
| Nginx | >= 1.20 | 反向代理（可选，也可直接访问 8000 端口） |

## 文件结构

```
server/
├── server.js          # 主服务程序
├── package.json       # 依赖配置
├── start.bat          # Windows 快速启动脚本
├── stop.bat           # Windows 停止脚本
├── logs/              # 运行日志（自动生成）
├── WorkSpace/         # 工作目录
└── node_modules/      # 依赖包（npm install 后生成）
```

## 安装步骤

### 1. 安装 Node.js

从 https://nodejs.org 下载 LTS 版本并安装，安装时勾选"Add to PATH"。

验证安装：
```bash
node -v
npm -v
```

### 2. 安装 MongoDB

从 https://www.mongodb.com/try/download/community 下载并安装。

安装后确认 MongoDB 服务已启动：
```bash
# 检查服务状态
net start | findstr MongoDB

# 或手动启动
net start MongoDB
```

默认连接地址：`mongodb://127.0.0.1:27017`

### 3. 部署服务

将 `server` 文件夹复制到目标目录，例如 `C:\NginxData\server`。

安装依赖：
```bash
cd C:\NginxData\server
npm install
```

### 4. 配置 Nginx（可选）

如需通过 80 端口访问，配置 Nginx 反向代理：

```nginx
server {
    listen 80;

    location / {
        root C:/NginxData;
        index index.html;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /users/ {
        proxy_pass http://127.0.0.1:8000;
    }
}
```

## 启动服务

按以下顺序启动：

### 方式一：使用启动脚本（推荐）
双击 `start.bat`，脚本会自动关闭旧进程并启动新服务。

### 方式二：命令行启动
```bash
cd C:\NginxData\server
node server.js
```

启动成功后会显示：
```
[init] MongoDB 已连接
[init] AI Image Studio 服务已启动: http://127.0.0.1:8000
```

### 启动顺序
1. MongoDB（必须先启动）
2. Node.js 后端（本服务）
3. Nginx（可选）
4. SD WebUI（可选，用于实际图片生成）

## 停止服务

### 方式一：使用停止脚本
双击 `stop.bat`

### 方式二：命令行
在运行窗口按 `Ctrl+C`，或执行：
```bash
cd C:\NginxData\server
stop.bat
```

## 关键配置项

配置位于 `server.js` 文件顶部：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `PORT` | 8000 | 服务监听端口 |
| `MONGO_URL` | mongodb://127.0.0.1:27017 | MongoDB 连接地址 |
| `MONGO_DB` | ai_image_studio | 数据库名称 |

队列限制配置存储在 MongoDB `config` 集合中：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `maxUserQueue` | 3 | 每用户最大排队数 |
| `maxGlobalQueue` | 50 | 全局最大排队数 |
| `maxImagesPerUser` | 20 | 每用户最大图片数 |
| `defaultMoney` | 50 | 新用户初始金币 |

## 常见问题

### Q: 启动报错 "MongoDB 连接失败"
确保 MongoDB 服务已启动：
```bash
net start MongoDB
```

### Q: 端口 8000 被占用
查找并关闭占用进程：
```bash
netstat -ano | findstr :8000
taskkill /F /PID <进程ID>
```

### Q: 图片生成一直排队
检查 SD WebUI Worker 是否已启动并连接到本服务。Worker 需要轮询 `GET /api/getImgQueue` 获取任务。

### Q: 手机浏览器显示"服务器未启动"
确保 Node.js 后端正在运行，且 Nginx 已正确配置反向代理。
